Technické detaily

Sdílená vrstva (styles.css)

  • Grid — 12 sloupcový, Bootstrap-konvence (.row, .col-md-4, .col-md-8 atd.), Latte-kompatibilní
  • Breakpoints — mobile-first; desktop ≥ 768px, small mobile ≤ 480px
  • Font — Open Sans 400/500/600/700 (Google Fonts)
  • Barevná paleta — CSS proměnné (--brand, --nav, --head, --tx, --mu, --bg, --s1, --s2, --line, --win, --loss)
  • Komponenty — match-row (res-row/prog-row), scoreboard, events, lineup, stat-table, form-dots, filter-bar, teams-grid, team-card, série, info-list, breadcrumb
  • Ikony — inline SVG sprite (zero HTTP requests), stroke-based
